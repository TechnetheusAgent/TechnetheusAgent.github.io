Vanquisher WR% Trajectory

How to use this chart

Interaction

Hover a line → focuses that character; shows its Mastery Gap and Meta Skew overlays.

Hover a legend name → previews a hidden character without toggling it on. Click the name to lock it visible (or hide it again).

Mouse drag → pans the time axis. Scroll wheel → zooms in/out (up to 10×). Double-click → resets zoom.

Filters

Hide All / Show All → clears the chart or restores everyone; useful for building a custom comparison.

Rising / Falling Trends → isolates characters the forecast says are climbing or sliding most.

X% Pred. / Obs. Outliers → top X% movers by forecast (Pred.) or recent history (Obs.). Adjust X% to widen/tighten.

Controls

Prediction Days → forecast horizon. 0 = history only.

Cones ≤ N chars → show per-character uncertainty bands when ≤ N characters are visible.

Daily WRs → overlay raw (unsmoothed) daily win-rate lines for visible characters.

Data source

Bucket → Vanquisher (top ~4%) or All-rank (everyone).

Weighting → Flat = every matchup equal (pure character strength). Pickrate = popular matchups weigh more (expected meta performance).

What the lines and numbers mean

Visual key

Solid line → smoothed historical win rate. Dashed line → forecast. Shaded cone → ±1 σ uncertainty. Thin dotted line → raw daily win rate (toggle via Daily WRs). White line (on hover) → Mastery Gap. Black line (on hover) → Meta Skew.

Metrics

Tier Score → average matchup win rate. 50% = balanced; above = favoured. Weighting mode controls how matchups combine.

Mastery Gap → Vanquisher WR minus All-rank WR. Positive = rewards skill. Negative = "pubstomper." Near zero = rank-agnostic.

Meta Skew → how lopsided the matchup spread is against popular opponents. Positive = meta bully. Negative = gets hard-countered by popular picks.

Forecast model

Blends momentum (recent trend continues) and mean-reversion (extremes pull toward 50%). Uncertainty cone grows then plateaus — volatile characters get wider bands.

Tooltip values: Rank = tier-score position at that date. PR# = pick-rate rank. (interp.) = data point was interpolated, not observed. drift = estimated daily WR change. σ = 1-step forecast uncertainty. NIS = normalized innovation squared (model-fit diagnostic; ~1 = well-calibrated). β = pick-rate shift coefficient. α = cross-bucket coupling (Vanq↔All). φ = mean-reversion strength.
